# 介绍

w-form是一款基于**uniapp+vue3**的**表单组件库**。

它相较传统表单组件会更加简洁方便，相较低代码组件又更加灵活。



##  安装

第一步、将插件代码并放入项目根目录下。

第二步、在`pages.json`中配置以下片段即可。

```json
  {
    ...
    "easycom": {
      "custom": {
        "^w-(.*)": "@/wform-components/components/w-$1/w-$1.vue"
      }
    }
  }
```
